<summary><b>⚙️ One-time setup for the metrics above (click to expand)</b></summary>
<br>

The two images above are generated by <code>.github/workflows/metrics.yml</code> using <a href="https://github.com/lowlighter/metrics">lowlighter/metrics</a>, running entirely on your own GitHub Actions — no dependency on a third-party server, so it won't go down or rate-limit like the old Vercel-hosted widgets did.

1. Create a **classic Personal Access Token**: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token, with scopes: `repo`, `read:user`, `read:org`.
2. In your `aloknayak24/aloknayak24` repo → Settings → Secrets and variables → Actions → New repository secret → name it `METRICS_TOKEN` → paste the token.
3. Commit the `.github/workflows/metrics.yml` file to this repo (already included).
4. Go to the **Actions** tab and manually run "Generate Profile Metrics" once (or wait for the daily 00:00 UTC cron).
5. It will commit `github-metrics.svg` and `github-isocalendar.svg` to the repo root, and the images above will pick them up automatically.

</details>
